## 🚀 Overview

**JARVIS AI** is a **modular, voice-enabled autonomous assistant** capable of:

* 🧠 Understanding natural language
* ⚙️ Executing system-level tasks
* 🌐 Performing intelligent web operations
* 💾 Maintaining contextual memory
* 🔄 Handling multi-step reasoning

It’s not just a chatbot —
👉 it’s a **mini AI operating system**.

---

## ⚡ Key Features

| Feature           | Description                        |
| ----------------- | ---------------------------------- |
| 🧠 AI Brain       | LLM-powered reasoning & planning   |
| 🎯 Intent Engine  | Hybrid NLP + AI classification     |
| 💻 System Control | Open/close apps, monitor system    |
| 🌐 Web Engine     | Smart search, scraping, automation |
| 💾 Memory System  | Context-aware conversations        |
| 🎤 Voice Input    | Speech recognition                 |
| 🔊 Voice Output   | Real-time TTS engine               |
| 🔌 Plugin System  | Extend functionality dynamically   |
| 📊 Logging        | Structured logs + debugging        |
| ⚡ Multi-Command   | Execute parallel/sequential tasks  |

## 🏗️ Architecture

```
User (Voice/Text)
      ↓
🧠 Assistant (Brain)
      ↓
⚙️ Command Center
      ↓
🎯 Intent Engine
      ↓
 ┌────────────┬────────────┬────────────┐
 │ 💻 System  │ 🌐 Web     │ 🤖 AI      │
 │ Automation │ Intelligence │ Reasoning │
 └────────────┴────────────┴────────────┘
      ↓
💾 Memory + 📊 Logger + ⚙️ Settings
      ↓
🔊 Response (Voice/Text)
```

## 📦 Tech Stack

* **Python** (Core system)
* **OpenAI API** (AI reasoning)
* **SpeechRecognition + Pyttsx3** (Voice I/O)
* **psutil + subprocess** (System control)
* **BeautifulSoup + requests** (Web scraping)
* **pytest** (Testing)

## ⚙️ Installation

```bash
git clone https://github.com/your-username/jarvis-ai.git
cd jarvis-ai
pip install -r requirements.txt
```

---

## 🔐 Setup

Create `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
VOICE_ENABLED=true
DEBUG=true
```

---

## ▶️ Run

### 🎤 Voice Mode

```bash
python run.py
```

### 💻 Text Mode (Recommended)

```bash
python run.py --mode text
```
